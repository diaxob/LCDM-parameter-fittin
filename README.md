# LCDM-parameter-fittin
Joint ΛCDM parameter fitting from SNe Ia μ(z), H(z) and BAO (D₍V₎/r_d). Python pipeline with custom Gauss–Newton &amp; Levenberg–Marquardt plus SciPy `least_squares`, hybrid Jacobians, column scaling, flatness prior, and bootstrap. Outputs plots and a JSON report. Full details in README.
